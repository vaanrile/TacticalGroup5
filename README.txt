Objectif du jeu : Eliminer toute l'équipe adverse.

Controle du jeu :
 - A et E : pour tourner la caméra
 - Espace : pour passer le tour de l'équipe
 - 1 : pour sélectionner le mode déplacement.
 - 2 : pour sélectionner le mode attaque.
 - 3 : pour sélectionner le mode première compétence.
 - 4 : pour sélectionner le mode deuxième compétence.
 - tabulation : pour sélectionner un autre personnage.


Classe :

 - mécano :
	Point de vie = 10
	Dégâts de base = 2
	Nombre de déplacement = 5
	Portée d'attaque = 4
	Précision de base = 75%
	Première compétence : Peut soigner d'une case adjacente un allier ou lui-même de 5 PV.

 - automate :
	Point de vie = 16
	dégâts de base = 5
	Nombre de déplacement = 4
	Portée d'attaque = 1,5
	Précision de base = 100%
	Première compétence : Charge sur une portée de 4 cases infligant au passage des dégâts.

 - char :
	Point de vie = 12
	Dégâts de base = 6
	Nombre de déplacement = 2
	Portée d'attaque = 6
	Précision de base = 66%
	Première compétence : Se met en mode tourelle ou en mode normal. Le mode tourelle l'empêche de se déplacer mais donne 2 de portée supplémentaires.

Quand on monte ou descend d'une case on prend 0,5 de mouvement en plus (cela implique aussi pour passer les tuyaux).
Pour attaquer un ennemi en mode d'attaque il suffit de cliquer sur l'ennemi qui est à la portée de la zone d'attaque.
